# О репозитории

Данный репозиторий является тестовым примером запуска в nanoCAD окна, созданного с помощью фреймворка Avalonia.

**Дисклеймер:** текущий вариант решения не является идеальным и конечным. В процессе появления новой информации репозиторий будет дополняться.

# Как запустить решение

1. Склонировать репозиторий. 
2. Открыть решение.
3. Проверить в NCadAvaloniaSample.Startup.csproj правильность пути к библиотекам nanoCAD API. По необходимости можно поменять путь к папке, где расположены библиотеки с API.
4. Проверить, что проект NCadAvaloniaSample.Startup назначен в качестве запускаемого.
5. Проверить в launchSettings.json или в свойствах отладки конфигурации Start in NCad путь к exe-шнику или файлу запуска nanoCAD. При необходимости поменять путь к точке запуска nanoCAD.
6. Собрать решение и запустить отладку.
